# Инструкция по установке APK

## Скачивание APK из GitHub Actions

1. Откройте https://github.com/vitalysennikov/laplog-app/actions
2. Найдите успешный workflow (с зеленой галочкой ✅)
3. Кликните на него
4. Прокрутите вниз до раздела **"Artifacts"**
5. Скачайте **"laplog-app"** (ZIP архив)
6. Распакуйте ZIP - внутри будет файл `app-release.apk`

## Установка на телефон

### Способ 1: Передача через USB

1. Подключите телефон к компьютеру через USB
2. Скопируйте `app-release.apk` в память телефона (например, в папку Downloads)
3. На телефоне откройте файловый менеджер
4. Найдите скачанный APK файл
5. Нажмите на файл для установки
6. Если появится запрос "Разрешить установку из неизвестных источников" - разрешите
7. Нажмите "Установить"

### Способ 2: Отправка по почте/мессенджеру

1. Отправьте `app-release.apk` себе на телефон (email, Telegram, WhatsApp и т.д.)
2. На телефоне откройте вложение
3. Скачайте APK
4. Откройте скачанный файл
5. Разрешите установку из неизвестных источников (если потребуется)
6. Нажмите "Установить"

### Способ 3: Через облачное хранилище

1. Загрузите APK в Google Drive, Яндекс.Диск или другое облако
2. На телефоне откройте ссылку на файл
3. Скачайте и установите

## Разрешение установки из неизвестных источников

Если при установке появляется сообщение о блокировке:

### Android 8.0 и выше:
1. Нажмите "Настройки" в диалоге
2. Включите "Разрешить из этого источника"
3. Вернитесь назад и повторите установку

### Android 7.1 и ниже:
1. Откройте Настройки → Безопасность
2. Включите "Неизвестные источники"
3. Вернитесь к установке APK

## Первый запуск

После установки:
1. Найдите иконку "LapLog Free" на главном экране или в списке приложений
2. Запустите приложение
3. Готово! Можно пользоваться секундомером

## Функции приложения

- **Start** - запуск секундомера
- **Pause** - пауза
- **Resume** - продолжение после паузы
- **Reset** - сброс на ноль
- **Lap** - добавить промежуточную отметку (доступно только при работающем секундомере)
- **Show milliseconds** (переключатель вверху) - показывать/скрывать миллисекунды

## Требования

- Android 7.0 (API 24) или выше
- ~5 МБ свободного места

## Обновления

Приложение не обновляется автоматически. Для установки новой версии:
1. Скачайте новый APK из GitHub Actions
2. Установите поверх существующего приложения (данные не потеряются)

## Удаление

Стандартным способом через настройки Android:
Настройки → Приложения → LapLog Free → Удалить
