Приложение для андроид - Секундомер.
Необходимо реализовать следующие функции:
1. Промежуточные отметки - время и длительность
2. Пауза
3. Возможность не отключать экран пока работает секундомер.
4. Темная тема
Формирование APK надо произвести в облаке с использованием GitHub Actions с минимальным скачиванием сред разработки на ПК.
Установку APK на телефон произведу самостоятельно.
